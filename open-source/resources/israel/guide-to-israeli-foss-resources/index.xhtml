Portals, News, etc.

  • The Israeli Linux Homepage

  • Hamakor - the Israeli Society for Free Software and Open Source Code - an NPO for FOSS in Israel.

  • whatsup.org.il - Hebrew Open Source News - including a news wire, some forums, a wiki, and an events calendar.

  • Linmagazine.co.il - a news/blogs site for Hebrew news.

  • Zulo - A wiki style site for enthusiasts of FOSS and technology - includes an audiocast, a videocast, comments, guides and documents and blogs. (Inactive).

  • Linux-Israel.Net

  • Hebrew Guides to Linux from guides.co.il

  • Planet FOSS-IL - a site aggregating Israeli Open Source Blogs.

  • “Halinkiyah” - a monthly developers’ newsletter in Hebrew by Ynon Perek.

Forums

  • Linux-related mailing-lists in Israel - index to the mailing lists, with subscription/unsubscription/post/archives information.

  • Whatsup Forums - various forums in Hebrew for questions.

  • Walla’s Linux Forum

  • Tapuz’s Linux Forum

  • Nana Linux Forum

  • Fresh’s Linux Forum

  • HWzone’s Linux Forum

  • Ubuntu Israel’s Forums

  • FXP's “Mac and Linux” forum

Clubs and Associations

  • Haifa Linux Club - bi-weekly meetings, online presentations material, a mailing list, job offers, links, etc.

  • Tel Aviv Open Source Club - meets every two weeks. The site includes the material for the past online presentations, as well as other information.

  • Jerusalem Linux Club - organises installation parties. Site inclues a wiki, forums, and other information.

  • Herzelinux - Linux in Herzeliyah - organises bi-weekly meetings.

  • Israeli Perl Mongers (Israel.pm or Perl-IL).

  • Python-IL - Israeli Python Users.

  • Ruby Israel (or Israel.rb).

  • Mozilla Israel (mozilla.org.il) - a site for the Mozilla project (Firefox, Thunderbird, etc.) in Israel - contains a wiki, a forum, a blog and news.

  • Israeli Association of Grid Technologies

  • Hebrew SUSE Linux Portal

  • Red Hat in Israel - a web-site for Linux in Israel and Red Hat in particular.

  • Debian Israel.

  • Ubuntu Israel.

  • Hackers-IL - a mailing list and a community site of software enthusiasts (= “Hackers”) dedicated to philosophical discussions of computers, technology, science and other topics.

  • Jerusalem Perl Mongers

  • Rehovot Perl Mongers

  • A Calendar for “All Linux Events” in Israel - currently lacks events from some clubs.

Meetup.com Groups

  • Open Source and Free Software in Israel - includes the FOSS Monthly meetup.

  • IL Tech Talks - Israeli tech talks.

  • Docker Tel Aviv

  • Ruby Underground Israel

  • PyWeb-IL - about the Python programming language primarily.

  • Django Israel - a group for the Django web framework.

  • TLV PHP Meetup - meetups in Tel Aviv about the PHP programming language.

  • React Israel

  • TLV Software Architecture Meetup

  • OpenStack Israel

  • Ansible Israel - about the Ansible platform for configuring and managing computers.

  • Modiin Tech Talks - tech talks in Modiin, Israel.

  • Sayeret Lambda - programming languages’ enthusiasts.

  • Underscore - the Israeli user group for Scala

  • Clojure Israel - about the Clojure programming language.

  • Full stack Developers Israel

  • Khronos Tel Aviv Chapter - about the Khronos Group standards.

Conferences

  • August Penguin - an annual community conference for FOSS in Israel.

  • Open Source Developers’ Conference in Israel - a conference for developers who use open-source technologies.

  • Welcome to Linux - an annual series of presentations introducing Linux to beginners.
