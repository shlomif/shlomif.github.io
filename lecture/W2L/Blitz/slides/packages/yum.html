7.2. בשימוש בפקודה yum

תוכןלמעלההקודםהבא
מבוא ללינוקס ← התקנת חבילות ← בשימוש בפקודה yum

כדי לעבור לנקודה הבאה או הקודמת בשקף יש ללחוץ במקלדת על החיצים שמאלה וימינה.

ניתן לעבור לשקף הבא על-ידי המקשים "N" ו-"P".

  • פדורה כוללת פקודת מימוש תלויות בשם "yum".
  • כדי להתקין חבילה יש להשתמש בפקודה "yum install [PACKAGE]".
  • לפעמים כדאי להוסיף את הדגל "-y" מיד אחרי הפקודה "yum" כדי שהתכנה לא תשאל שאלות מיותרות.
  • ניתן למצוא חבילות באמצעות הפקודה "search".
