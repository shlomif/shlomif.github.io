5.2. Command Mode and Editing Mode

ContentsUpPrevNext
Linux' Basic Use → Editing with Vi → Command Mode and Editing Mode
  • Vi has two modes of operation: command mode and editing mode. Vi starts in command mode.
  • In editing mode, one can insert new text.
  • In command mode, one can enter a myriad of commands, including ones to save the file, or to delete text.
