3.1. The Man Pages

ContentsUpPrevNext
Linux' Basic Use → Getting Help → The Man Pages
  • A Man page is a manual that covers a certain Linux command.
  • To access the listing of man pages, open a KDE web-browser and type man:/ at the Location display.
  • For Linux commands, you'll be interested in Section 1.
  • To access a man page for a particular command type "man:/[Command Name]" at the Location display.
  • To access a man page from the command line type man [command name]
