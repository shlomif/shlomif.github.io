3.3. The "Info" manuals

ContentsUpPrevNext
Linux' Basic Use → Getting Help → The "Info" manuals
  • The Info manuals are detailed guides that describe certain commands and applications.
  • To access them type info:/[cmd name] at the location entry of a KDE browser.
  • For instance info:/gcc will display the manual of "gcc".
