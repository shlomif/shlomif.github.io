3.5. Internet Resources

ContentsUpPrevNext
Linux' Basic Use → Getting Help → Internet Resources
  • There are numerous mailing-lists for various Linux subjects. A list of Israeli ones can be found here: http://www.iglu.org.il/mailing-lists/ .
  • Before posting to one it is recommended to refer to the How to Ask Questions the Smart Way FAQ.
  • Many times searching the web using web engines will yield good information. Google and Google Groups are especially very good.
  • Remember to RTFM (Read the Fabulous Manual) and STFW (Search the Fabulous Web) before you ask a mailing list.
