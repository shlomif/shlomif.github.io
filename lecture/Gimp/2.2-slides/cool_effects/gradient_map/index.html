6.1. Gradient Map

ContentsUpPrevNext
Do it with the GIMP → Some Cool Effects → Gradient Map
  • A gradient map maps each pixel to an RGB triad found in the active gradient in proportion to its brightness level.
  • It is in fact a V → RGB LUT in which the resultant colors are chosen arbitrarily according to the gradient.
