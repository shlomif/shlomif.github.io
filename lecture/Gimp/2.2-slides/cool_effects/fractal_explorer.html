6.2. Fractal Explorer

ContentsUpPrevNext
Do it with the GIMP → Some Cool Effects → Fractal Explorer
  • The fractal explorer is a GIMP plug-in that can be used to render some fractals.
  • It supports only a subset of all the known two-dimensional fractals, but it is quite powerful at rendering them.

Demo

  1. Create an empty image.
  2. Invoke the Fractal Explorer with <Image> → Filters → Render → "Fractal Explorer..."
  3. Play with it. (the upper-left preview is zoomable)
