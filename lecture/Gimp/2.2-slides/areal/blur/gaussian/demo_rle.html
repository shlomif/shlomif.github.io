4.1.3.3. Demo of RLE Gauss. Blur

ContentsUpPrevNext
Do it with the GIMP → Basic Areal Effects → Blur → Gaussian Blur → Demo of RLE Gauss. Blur

Demo

  1. Create a blank white image.
  2. Place a black text caption on it using the text tool.
  3. Merge the visible layers.
  4. Select <Image> → Filters → Blur → "Gaussian Blur..."
  5. Select radii of x = 5 and y = 5.
  6. Select "RLE" for the Blur Method.
  7. Press the "OK" Button.
