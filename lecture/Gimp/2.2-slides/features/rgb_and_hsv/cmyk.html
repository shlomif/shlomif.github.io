2.3.3. What about CMYK?

ContentsUpPrevNext
Do it with the GIMP → Overview of Features → RGB and HSV modes → What about CMYK?
  • CMYK stands for Cyan, Magenta, Yellow and blacK.
  • It is a subtractive method that is akin to mixing paint colours.
  • One can select colours using it by using the Printer-iconed tab of the colour selector.
  • Nevertheless, GIMP 2.2 lacks full support for CMYK.
