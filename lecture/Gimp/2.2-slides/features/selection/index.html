2.6. Selection and its Usage

ContentsUpPrevNext
Do it with the GIMP → Overview of Features → Selection and its Usage
  • If you define a selection, then only the pixels of the selection are affected by tools or filters.
  • GIMP Selections can be any subset of pixels.
  • The toolbox contains numerous selection tools that can be used to shape the selection.
  • Pressing Ctrl, Alt and Shift before pressing the button and while releasing it modifies the behaviour of the selection.
  • Just experience to see how everything works!

Demo

  1. Open a new image
  2. Select the "Select Rectangular areas" tool
  3. Play with it (try pressing the Ctrl, Alt and Shift keys when selecting, too).
  4. Try other tools and see how they affect the selection.
