2.6.1. Using the Red "Paint Selection" Button

ContentsUpPrevNext
Do it with the GIMP → Overview of Features → Selection and its Usage → Using the Red "Paint Selection" Button
  • An image has a mode that enables you to "paint" the selection using existing tools.
  • In this mode colouring with black adds pixels to the selection.
  • And colouring with white excludes them.
  • To access this mode press the button in the lower-left corner of the image, right to the left of the bottom scrool-bar.
  • Note: this mode has a meta-selection, but there aren't any metaN-selections.

Demo

  1. Create a new image
  2. Select some region in it.
  3. Press button
  4. Switch colours from black to white and paint the selection.
  5. Switch back to Selection Mode
