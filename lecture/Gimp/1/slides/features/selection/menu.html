2.6.2. The "<Image> -> Select" Menu

ContentsUpPrevNext
Do it with the GIMP → Overview of Features → Selection and its Usage → The "<Image> -> Select" Menu
  • The <Image> -> Select sub-menu is handy in performing several common-operations on selections.
  • Among the tools that can be found are:
    1. Select None
    2. Select All
    3. Invert the Selection
    4. Sharpen the Selection
