7.2. Gimpressionist

ContentsUpPrevNext
Do it with the GIMP → Turning a Photograph into a Painting → Gimpressionist
  • The Gimpressionist takes an image and paints it for you using a specified brush and many other parameters.

Demo

  1. Open g_anwar1.jpg
  2. Select <Image> -> Filters -> Artistic -> "Gimpressionist..."
  3. Select a preset and press the "Apply button"
  4. Play with the settings in the different tabs if you don't like the preview.
  5. Press the OK Button.
  6. Undo and do it with other images and/or settings.
