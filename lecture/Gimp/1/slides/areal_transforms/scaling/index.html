5.1. Scaling an Image

ContentsUpPrevNext
Do it with the GIMP → Basic Areal Transformations → Scaling an Image
  • In order to scale an image use the <Image> -> Image -> "Scale Image..." menu option.

Demo

  1. Open tiger_sitting.jpg
  2. Select <Image> -> Image -> "Scale Image..."
  3. Increase it by 3 in each dimension. (you can lock them together)
